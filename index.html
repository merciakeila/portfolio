Ferramentas que eu uso

Adobe Photoshop

É um software caracterizado como editor de imagens,usado atualmente por muitas empresas de diversos ramos, e também por revistas, fotógrafos e na publicidade.

Adobe Illustrator

É um editor de imagens vetoriais, ele permite criar ilustrações, gráficos, logos e muito mais para ser utilizado em qualquer tamanho e mídia.

Adobe Premiere

É o principal software de edição de vídeo para cinema, TV e a Web. Ferramentas de criação, integração com outros aplicativos e serviços da Adobe.

Adobe After Effects

É um programa de criação de gráficos com movimento e efeitos visuais

Adobe InDesign

Desenvolvido para diagramação e organização de páginas.

CorelDRAW

É um programa que possibilita a criação e a manipulação conteúdos gráficos.

HTML,CSS & JavaScript

É a base para a programação Front-End, com elas de forma tão uniforme, que vai garantir que o site criado seja super atrativo e de fácil leitura em todos os lugares: web, smartphone, tablete, entre outros canais onde o site puder ser aberto pelos navegadores.

Bootstrap

É um framework web com código-fonte aberto para desenvolvimento de componentes de interface e front-end para sites e aplicações web usando HTML, CSS e JavaScript, baseado em modelos de design para a tipografia, melhorando a experiência do usuário em um site amigável e responsivo.
